Hugo 日常使用指南

一、基本操作

  1. 创建新文章
# 创建新文章
hugo new posts/my-new-post.md
# 或者指定类型
hugo new --kind post posts/my-new-post.md
  1. 本地预览
# 启动本地服务器
hugo server -D  # -D 参数会显示草稿

# 指定端口启动
hugo server -D --port 1313

# 允许局域网访问
hugo server -D --bind 0.0.0.0
